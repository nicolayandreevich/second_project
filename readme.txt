Исследовательский проект по прогнозированию оттока клиентов из банка. 
Необходимо создать модель, которая будет предсказывать уход клиента из банка.

Модели, использованные для прогназирования следующие:
1) DecisionTreeClassifier
2) RandomForestClassifier
3) LogisticRegression

Результат:
Модель RandomForestClassifier сомогла довести метрику F1 до значения 0.6, ROC_AUC до 0.84
